## 🎨 模板清单

### 一、服饰文字排版（6 个模板）

专注于服装印花中的文字排版设计，适合标语、品牌名、态度表达等场景。

#### 1. 街头大字 `streetwear-bold-text`
- **难度**: ⭐ 简单
- **适用产品**: T恤、卫衣、帆布包、帽子
- **场景**: 街头文化、态度表达、年轻人市场
- **特点**: 超大粗体无衬线字体，视觉冲击力强
- **文案建议**: 2-6 个词，大写效果最佳

```
示例文案：
主标语: STAY WILD
副标语: Est. 2024
```

#### 2. 极简衬线名言 `minimal-serif-quote`
- **难度**: ⭐ 简单
- **适用产品**: T恤、帆布袋、明信片、笔记本封面
- **场景**: 文艺青年、复古风格、知识分子风
- **特点**: 优雅衬线字体，经典引用格式
- **文案建议**: 名言、歌词、诗句，3-15 个词

```
示例文案：
名言: "Less is more."
作者: Mies van der Rohe
```

#### 3. 复古徽章 `retro-vintage-badge`
- **难度**: ⭐⭐ 中等
- **适用产品**: T恤、帽子、贴纸、马克杯
- **场景**: 品牌周年庆、复古主题、工装风格
- **特点**: 弧形文字 + 中心图形，仪式感强
- **文案建议**: 品牌名、地点、年份等信息

```
示例文案：
上方: AUTHENTIC
中心: BRAND
下方: SINCE 1990
年份: 1990
```

#### 4. 现代极简品牌 `modern-minimalist-brand`
- **难度**: ⭐ 简单
- **适用产品**: T恤、卫衣、托特包、手机壳
- **场景**: 新品牌发布、简约系列、品牌形象
- **特点**: 干净利落，现代感强，不会过时
- **文案建议**: 品牌名或系列名，简短标语

```
示例文案：
品牌名: STUDIO.
标语: Design & Craft
```

#### 5. 日式禅意 `japanese-zen-text`
- **难度**: ⭐ 简单
- **适用产品**: T恤、帆布袋、茶杯、笔记本
- **场景**: 日本文化、禅意生活、极简主义
- **特点**: 大量留白，意境深远
- **文案建议**: 1-4 个汉字，有意境的词

```
示例文案：
汉字: 侘寂
读音: WABI-SABI
描述: Beauty in imperfection
```

#### 6. 学院运动风 `college-varsity-text`
- **难度**: ⭐⭐ 中等
- **适用产品**: 卫衣、T恤、帽子、运动包
- **场景**: 学校社团、毕业纪念、校队队服
- **特点**: 美式大学运动风格，辨识度高
- **文案建议**: 学校名、社团名、入学年份

```
示例文案：
学校: UNIVERSITY
社团: ATHLETICS
年份: 2024
```

### 二、电商营销（5 个模板）

专注于电商产品展示、促销活动、品牌营销等场景。

#### 1. 产品促销横幅 `product-promo-banner`
- **难度**: ⭐ 简单
- **适用产品**: 产品包装插卡、电商详情页海报、社交媒体广告
- **场景**: 黑色星期五、季节性清仓、新品上市
- **特点**: 强烈色彩对比，大字号突出折扣
- **文案建议**: 折扣信息 + 活动名称 + 行动呼吁

```
示例文案：
折扣: 50% OFF
活动: SUMMER SALE
详情: Limited time offer. Shop now!
```

#### 2. 产品展示卡片 `product-feature-card`
- **难度**: ⭐⭐ 中等
- **适用产品**: 产品包装插卡、电商主图、社交媒体产品帖
- **场景**: 电商产品展示、社交媒体推广
- **特点**: 清晰的层级关系，专业展示
- **需要**: 白底产品图

```
示例文案：
名称: Premium T-Shirt
描述: High-quality cotton. Comfortable fit.
价格: $29.99
```

#### 3. 优惠券/代金券 `coupon-voucher`
- **难度**: ⭐ 简单
- **适用产品**: 优惠券打印、礼品卡、邮件营销附件
- **场景**: 会员专享、节日优惠、新用户奖励
- **特点**: 虚线切割设计，增加仪式感
- **文案建议**: 优惠信息 + 优惠码 + 使用条款

```
示例文案：
优惠: $10 OFF
优惠码: SAVE10
条款: Valid until Dec 31, 2024
```

#### 4. 礼品卡 `gift-card`
- **难度**: ⭐⭐ 中等
- **适用产品**: 实体礼品卡、电子礼品卡、节日礼盒卡片
- **场景**: 节日送礼、会员积分兑换、品牌活动
- **特点**: 精致设计，增加礼品仪式感
- **文案建议**: 品牌名 + 面值 + 祝福语

```
示例文案：
品牌: STUDIO
面值: $100
祝福: Thank you for being amazing!
```

#### 5. 价格标签/贴纸 `price-tag-sticker`
- **难度**: ⭐ 简单
- **适用产品**: 产品价签、SKU 贴纸、库存管理标签
- **场景**: 实体店标价、电商 SKU 管理
- **特点**: 清晰易读，可批量打印
- **文案建议**: 价格 + 产品名称 + SKU 编号

```
示例文案：
价格: $39.99
产品: T-Shirt
SKU: TSH-001
```

### 三、社交媒体（5 个模板）

专为 Instagram、Twitter/X、Facebook 等社交平台设计。

#### 1. Instagram 帖子 `instagram-square-post`
- **难度**: ⭐ 简单
- **适用产品**: Instagram 帖子、Facebook 帖子、LinkedIn 图片
- **场景**: 产品推广、品牌故事、用户见证
- **特点**: 1:1 正方形，最通用的 Instagram 格式
- **需要**: 高质量产品图或生活方式照片

```
示例文案：
标题: New Collection
正文: Discover our latest arrivals.
账号: @yourbrand
```

#### 2. Instagram Story 竖屏 `instagram-story`
- **难度**: ⭐⭐ 中等
- **适用产品**: Instagram Story、TikTok 封面、Snapchat 广告
- **场景**: 限时动态、活动预告、倒计时
- **特点**: 9:16 竖屏，最适合手机用户
- **文案建议**: 大标题 + 副标题 + 行动按钮

```
示例文案：
标题: FLASH SALE
副标题: 24 hours only. Don't miss out!
按钮: Shop Now →
```

#### 3. Twitter/X 引用图 `twitter-x-post`
- **难度**: ⭐ 简单
- **适用产品**: Twitter/X 帖子、LinkedIn 图片、博客配图
- **场景**: 分享名言、数据、观点、小贴士
- **特点**: 带图片的帖子转发率更高
- **文案建议**: 名言、数据或观点，简短有力

```
示例文案：
引用: Design is not just what it looks like...
作者: — Steve Jobs
```

#### 4. 用户评价卡片 `testimonial-card`
- **难度**: ⭐⭐ 中等
- **适用产品**: 社交媒体证明、电商详情页、官网落地页
- **场景**: 客户反馈、产品评价、服务体验
- **特点**: 增加品牌信任度
- **需要**: 用户头像（可选）

```
示例文案：
评价: This product changed my life...
姓名: Sarah M.
身份: Verified Buyer
评分: 5 星
```

#### 5. 公告/徽章 `announcement-badge`
- **难度**: ⭐ 简单
- **适用产品**: 社交媒体公告、邮件头图、网站横幅
- **场景**: 新产品上线、获得奖项、用户里程碑
- **特点**: 圆形徽章格式，醒目且正式
- **文案建议**: 标签 + 主要公告 + 副标题

```
示例文案：
标签: ANNOUNCING
主要: NEW PRODUCT
副标题: Coming Soon
```

## 🎯 模板使用指南

### 选择合适的模板

1. **卖服装？** → 选择「服饰文字排版」系列
   - 街头潮牌：街头大字、学院运动风
   - 文艺复古：极简衬线名言、日式禅意
   - 品牌形象：现代极简品牌、复古徽章

2. **做电商？** → 选择「电商营销」系列
   - 促销活动：产品促销横幅、优惠券
   - 产品展示：产品展示卡片、价格标签
   - 礼品赠送：礼品卡

3. **运营社媒？** → 选择「社交媒体」系列
   - Instagram：帖子、Story 竖屏
   - Twitter/X：引用图
   - 品牌建设：用户评价、公告徽章

### 自定义模板

每个模板都有可编辑的变量：
- **文字变量**: 修改文案内容
- **颜色变量**: 调整品牌色
- **图片变量**: 替换产品图
- **字体变量**: 选择适合的字体
- **数字变量**: 调整间距、大小等

### 导出使用

1. 选择模板
2. 编辑变量（文案、颜色、图片）
3. 实时预览效果
4. 导出 PNG 或上传到素材库
5. 应用到产品或社交媒体

## 💡 最佳实践

### 文案撰写
- **简短有力**: 社交媒体 3-5 秒阅读时间
- **行动呼吁**: 使用 "Shop Now"、"Learn More" 等
- **情感共鸣**: 用情绪化文案打动用户
- **数据支撑**: "50% OFF" 比 "Big Sale" 更有效

### 配色建议
- **品牌一致**: 使用品牌主色和辅助色
- **对比强烈**: 背景和文字要有足够对比度
- **情绪匹配**: 
  - 红色/橙色：紧迫感、促销
  - 蓝色：信任、专业
  - 绿色：自然、健康
  - 黑色/金色：高端、奢华

### 图片选择
- **高质量**: 至少 1080x1080 像素
- **主体明确**: 焦点清晰，背景简洁
- **风格统一**: 保持品牌视觉一致性
