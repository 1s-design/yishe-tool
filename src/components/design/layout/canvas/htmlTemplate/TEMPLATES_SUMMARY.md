# 新增 HTML 模板总结

## 🎯 设计理念

重新设计的模板遵循以下原则：

1. **专业性** - 每个模板都经过精心设计，有设计感

## 🚀 如何使用

1. 打开 HTML 元素编辑器
2. 点击"打开模板库"
3. 使用筛选条件快速定位：
   - 分类：服饰/电商/社交媒体
   - 难度：简单/中等
   - 印花方式：cutout/poster/label
4. 点击应用模板
5. 编辑变量（文案、颜色、图片）
6. 实时预览效果
7. 保存并导出
